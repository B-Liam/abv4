Planning Vitamin C

Vitamin C

Vegetables and fruit are excellent sources of vitamin C. Vitamin C supports the health of cells and tissue, the healing of wounds, and generally helps the body fight infection. Vitamin C cannot be stored in the body, so eating foods containing the vitamin on a daily basis is recommended.
  • Helps keep cells healthy
  • Helps wounds heal
  • Helps avoid scurvy
